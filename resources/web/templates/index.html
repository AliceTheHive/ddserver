Welcome to ddserver

About

This application allows you to specify hostnames in a dynamic DNS zone. It can be used, to set a domain name for dynamic IP addresses, allowing you to access those addresses by name, even if the IP address changes.

The update protocol of this application is compatible to the dyndns2 protocol, allowing you to use your homerouters DDNS service or a tool like ddclient to send updates of your address.
