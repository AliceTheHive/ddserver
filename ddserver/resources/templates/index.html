Welcome to ddserver

ddserver is a server-side application for dynamic DNS management.

It allows you to specify hostnames (subdomains) inside a DNS zone, and to update the IPv4 address of these hostnames dynamically by using a simple update protocol over HTTP(S). This enables you to access hosts with changing IPv4 addresses by a static domain name. As updates of the IP address are performed using an open update protocol, most DSL home-routers are able to send IP address updates. Also, there is a tool called ddclient, which can be used to send updates from any *NIX-based operating system. Additionally, a static IPv6 address can be assigned for each hostname.

Documentation and Support

For further documentation, please refer to http://docs.ddserver.org.
